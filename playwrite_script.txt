PlayWright Agents:
Three Specialized Agents 
Planner (creates test, plans) 
Understands your app structure 
Uses seed files and utilities 
Respects existing code patterns 
Foundation for good tests
Generator (writes tests based on plans) 
One test per file (recommended)
Verifies elements before writing 
Uses test logs to prevent hallucination
Runs multiple times for accuracy 
Healer (fixes broken tests) 
Runs tests in debug mode
Read console errors 
Checks API responses 
Adds ‘FIXME’ for app bugs
